# 思路一

双指针，分别指向奇数位和偶数位。

偶数位从左向右遍历，遇到奇数值需要奇数位偶数值的元素，两者替换。

``` js
/**
 * @param {number[]} A
 * @return {number[]}
 */
var sortArrayByParityII = function(A) {
    let even = 0;
    let odd = 1;
    const size = A.length;

    while (even < size) {
        if (A[even] % 2 === 1) {
            while (A[odd] % 2 === 1) {
                odd += 2;
            }
            [A[odd], A[even]] = [A[even], A[odd]];
        }
        even += 2;
    }

    return A;
};
```
