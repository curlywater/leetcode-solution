# 题目描述

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例:

```
给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为：
[
  [-1, 0, 1],
  [-1, -1, 2]
]
```

# 解题思路

准备一个map用于存放值和索引对应关系。遍历nums数组，遍历到一个整数时，确认map中是否存在拼凑目标值的另一个整数。存在则返回两个整数的索引，不存在则把当前整数和索引存入map。时间复杂度$O(n)$
