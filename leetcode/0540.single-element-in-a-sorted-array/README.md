# 题目描述
给定一个只包含整数的有序数组，每个元素都会出现两次，唯有一个数只会出现一次，找出这个数。

示例:
```
输入: [1,1,2,3,3,4,4,8,8]
输出: 2
```

# 解题思路

中点的情况：
1. 右边长度是奇数（当前是奇数索引）
   1. 与左边相等，目标数在右区间
   2. 不与左边相等，目标数是本身或在左区间
2. 右边长度是偶数（当前是偶数索引）
   1. 与右边相等，目标数在右区间
   2. 不与右边相等，目标数是本身或在左区间


优化，两种情况可以通过索引奇偶性转换，压缩为一种情况。
