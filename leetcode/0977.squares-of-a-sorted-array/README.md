# 题目描述

给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

示例:

```
输入：[-4,-1,0,3,10]
输出：[0,1,9,16,100]
```

说明：
- 1 <= A.length <= 10000
- -10000 <= A[i] <= 10000
- A 已按非递减顺序排序。



来源：力扣（LeetCode）

链接：https://leetcode-cn.com/problems/squares-of-a-sorted-array/

# 解题思路

负数到正数平方的特征是先减后增，呈山谷状态。

利用该特性进行按非递减顺序排序，容易想到从两端向中间推进。
