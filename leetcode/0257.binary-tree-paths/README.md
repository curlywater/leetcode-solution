# 题目描述
给定一个二叉树，返回所有从根节点到叶子节点的路径。

说明: 叶子节点是指没有子节点的节点。

示例:
```
输入:

   1
 /   \
2     3
 \
  5

输出: ["1->2->5", "1->3"]

解释: 所有根节点到叶子节点的路径为: 1->2->5, 1->3
```

来源：力扣（LeetCode）

链接：https://leetcode-cn.com/problems/binary-tree-paths

# 解题思路

深度优先遍历，但存在是否使用回溯的两种写法。区别在于：不使用回溯需要在每次递归时传入复制的字符串，使用回溯可以使用全局的数组保存状态，回溯的优点之一直接修改共用状态。
